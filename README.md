# QMusic
本项目是为了完成Java大作业而做的安卓本地音乐播放器，主要实现的功能有：

1. 扫描本地音频文件
2. 列表展示所有音乐
3. 点击列表内的音乐进行播放
4. 播放控制（播放、暂停、进度条拖动）

项目开源于https://github.com/Fangnan700/QMusic
